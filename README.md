**Super Organizer** — это мощное веб-приложение "Всё-в-одном" для организации личной жизни. Проект объединяет 6 полезных инструментов в едином интерфейсе с единой системой авторизации.

Главная особенность — **продвинутый движок тем**, включающий секретный режим "Гарри" с анимированным фоном, эффектом стекла (Glassmorphism) и звуковым сопровождением.

---

## ✨ Основные возможности

### 🛠 6 Встроенных модулей
Приложение заменяет собой 6 разных программ:
1.  **📝 Заметки:** Быстрое создание и удаление личных записей.
2.  **🛒 Список покупок:** Добавление продуктов с количеством, чекбоксы (куплено/нет), сортировка (купленные уходят вниз).
3.  **📞 Телефонная книга:** Хранение контактов с **живым поиском** по имени.
4.  **🖊 Мини-блог:** Публикация мыслей с автоматической датой и временем.
5.  **🎬 Каталог фильмов:** Карточки фильмов с названием, жанром, годом и цветным рейтингом.
6.  **✅ ToDo Лист:** Задачи с дедлайнами и статусами выполнения.

### 🎨 Уникальная система тем
Используется `CSS Variables` и `LocalStorage` для мгновенного переключения стилей без перезагрузки:
* **☀ Светлая тема:** Чистый минимализм, высокая читаемость.
* **🌙 Темная тема:** Глубокие цвета, снижает нагрузку на глаза.
* **🐱 Режим "Гарри" (Секретный):**
    * **Живые обои:** Плавное появление фона.
    * **Glassmorphism:** Полупрозрачные карточки с размытием фона (backdrop-filter).
    * **Анимация:** Эффект "Ocean Waves" (живой градиент) на всех элементах интерфейса.
    * **Атмосфера:** При активации включается звук морского прибоя (Audio API).

## ⚙️ Технологический стек

* **Backend:** Python 3, Flask.
* **Database:** SQLite3 (7 связанных таблиц).
* **Frontend:** HTML5, CSS3 (Flexbox, Grid, Animations), Vanilla JS.
* **Security:** Хэширование паролей (`werkzeug.security`), защита сессий, защита от XSS (Jinja2 escaping).

## 🚀 Установка и запуск

Следуйте инструкции, чтобы развернуть проект на своем компьютере.

### 1. Клонирование репозитория

```bash
git clone [https://github.com/ВАШ_НИК/flask-super-organizer.git](https://github.com/ВАШ_НИК/flask-super-organizer.git)
cd flask-super-organizer

```

### 2. Создание виртуального окружения (Рекомендуется)

```bash
# Linux / MacOS
python3 -m venv .venv
source .venv/bin/activate

# Windows
python -m venv .venv
.venv\Scripts\activate

```

### 3. Установка зависимостей

```bash
pip install -r requirements.txt

```

### 4. Запуск

```bash
python app.py

```

*При первом запуске приложение автоматически создаст базу данных `superapp.db` и все необходимые таблицы.*

Перейдите в браузере по адресу: `http://127.0.0.1:5000`
