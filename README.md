# 🏝️ Flask Super Organizer

Многофункциональное веб-приложение "Всё-в-одном" с уникальной системой тем.

## 🚀 Функционал
* **6 Модулей:** Заметки, Список покупок, Контакты, Блог, Фильмы, ToDo.
* **Темы оформления:** Светлая, Темная и Секретная тема "Гарри".
* **Атмосфера:** В теме "Гарри" включается анимированный фон (волны), эффект стекла (Glassmorphism) и звук океана.
* **Технологии:** Flask, SQLite, CSS Variables, JS Audio API.

## 🛠 Установка
1. Клонировать репозиторий.
2. `pip install -r requirements.txt`
3. `python app.py`
